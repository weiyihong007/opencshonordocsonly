# OpenCSHonor:计试一站式学术资源共享项目
**OpenCSHonor**是一个学术资源共享项目,旨在精心组织计试的所有资料以便于在线访问,本项目将由计学组网站组同学**长期维护**。

我们的核心目标是使得**OpenCSHonor**成为计试的同学从大一入学到大四毕业寻找学术资源的**第一选择**。
